# 🌌 Kodari Universe: AI Micro-SaaS Suite

Professional AI-powered tools suite for social media marketers, job seekers, and financial management.

## 📦 Features
- **SNS-Spark**: AI Instagram Caption Generator with Sentiment Analysis.
- **InterviewAce**: AI-driven Interview Preparation Assistant (Multi-language).
- **QuickInvoice**: Automated Receipt Scanning & Expenditure Dashboard.
- **Unified Auth**: Shared login/logout system via `auth.js`.
- **AdSense Ready**: Optimized slots and `ads.txt` for monetization.

## 🚀 How to Deploy

### 1. GitHub Upload
1. Create a new repository on GitHub (e.g., `kodari-universe`).
2. Run the following commands:
   ```bash
   git init
   git add .
   git commit -m "🚀 Initial Launch: Kodari Universe Suite"
   git remote add origin https://github.com/YOUR_USERNAME/kodari-universe.git
   git push -u origin main
   ```

### 2. Vercel Deployment
1. Go to [Vercel](https://vercel.com/new).
2. Import the `kodari-universe` repository.
3. Vercel will automatically detect the static files. Click **Deploy**.

### 3. Cloudflare Pages Deployment
1. Go to the [Cloudflare Dashboard](https://dash.cloudflare.com/).
2. Navigate to **Workers & Pages** -> **Create application** -> **Pages** -> **Connect to Git**.
3. Select the repository and deploy.

## 💰 Monetization
- Replace the `pub-0000...` ID in `ads.txt` with your actual AdSense Publisher ID.
- Add your AdSense script tag to the `<head>` of `index.html`.
